Some Extra Attributes and Types of Lists

This next part will go over some of the extra attributes and Types of lists that exists. These are:

  • Start
  • Reversed
  • Value
  • Description Lists
Start

This an example of using Start in a ordered list.

  1. This is the first list item
  2. This is the second list item
  3. This is the third list item
Reveresed

This is an example of using Revered in an ordered list.

  1. This is three
  2. This is two
  3. This is one
Value

This is an example of using Value in an ordered list.

  1. This list item has a value of 1.
  2. This list item has a value of 6.
  3. This list item has a value of 15.
  4. This list item has a value of 16.
